# ⚾ 컴투스 프로야구 V25 - 랭킹챌린지 기록 관리

컴투스 프로야구 V25 랭킹챌린지의 **요일별 선수 기록을 입력·관리**하는 웹 앱입니다.  
별도 서버나 설치 없이 HTML 파일 하나로 바로 실행됩니다.

---

## 🚀 실행 방법

### 방법 1 — 파일 직접 열기 (가장 간단)

`index.html` 파일을 **더블클릭**하거나,  
Chrome / Edge / Firefox 브라우저로 드래그&드롭하면 바로 실행됩니다.

```
comtus_rankingchallenge/
├── index.html   ← 이 파일을 브라우저로 열기
├── style.css
└── app.js
```

### 방법 2 — VS Code Live Server 사용

1. VS Code에서 `Live Server` 확장 설치
2. `index.html` 파일을 열고 우하단 **Go Live** 클릭
3. 브라우저에서 자동으로 열림

### 방법 3 — 로컬 HTTP 서버 (Python)

```bash
# 프로젝트 폴더에서 실행
python -m http.server 8080

# 브라우저에서 접속
# http://localhost:8080
```

## 📋 주요 기능

| 기능 | 설명 |
|------|------|
| 📅 요일별 탭 | 월~일 각 요일마다 독립적으로 기록 관리 |
| 📊 누적기록 | 전 요일 기록을 자동으로 합산하여 표시 |
| 🏏 타자 기록 | 9개 포지션 (포수·1루수·2루수·3루수·유격수·좌익수·중견수·우익수·지명타자) |
| ⚾ 투수 기록 | 10개 포지션 (SP1~SP5·RP1~RP4·CP) |
| 🔢 자동 계산 | OPS·출루율·장타율·타율·득점권타율·ERA 실시간 자동 계산 |
| 💾 자동 저장 | 브라우저 localStorage에 저장 (새로고침해도 유지) |
| 📋 CSV 내보내기 | 전 요일 기록을 CSV 파일로 저장 |

## 🏏 타자 입력 항목

`경기` · `타수` · `안타` · `2루타` · `3루타` · `홈런` · `볼넷` · `희생플라이` · `득점권타수` · `득점권안타`

> OPS · 출루율 · 장타율 · 타율 · 득점권타율은 입력값을 바탕으로 자동 계산됩니다.

## ⚾ 투수 입력 항목

`경기` · `승` · `패` · `이닝` · `자책점`

> ERA는 자동 계산됩니다. 이닝 입력 시 `7`, `69 1/3`, `147 2/3` 형식으로 입력하세요.

---

## 💡 사용 팁

- **탭 전환 시 자동 저장** 됩니다.
- **누적기록** 탭은 읽기 전용으로, 모든 요일의 합산 결과를 보여줍니다.
- 데이터를 완전히 지우려면 우측 상단 **🗑 초기화** 버튼을 사용하세요.
- **📋 내보내기** 버튼으로 전체 기록을 CSV 파일로 저장할 수 있습니다.
